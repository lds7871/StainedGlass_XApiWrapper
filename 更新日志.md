## 更新日志

- **26/3/31（分支: copilot/upgrade-jdk25-spring4-optimization）**：
  - 升级 JDK 版本: Java 17 → Java 25 (LTS)。
  - 升级 Spring Boot 版本: 3.1.4 → 4.0.1（基于 Spring Framework 7）。
  - 替换 Springfox Swagger 2 → SpringDoc OpenAPI 3（springdoc-openapi-starter-webmvc-ui 2.8.0），全面迁移 `@Api`/`@ApiOperation`/`@ApiModel`/`@ApiModelProperty` 等注解至 OpenAPI v3 标准注解。

  - 更新 MySQL 驱动: `mysql:mysql-connector-java:8.0.33` → `com.mysql:mysql-connector-j:9.1.0`。
  - 更新 Lombok: 1.18.30 → 1.18.36；FastJSON2: 2.0.40 → 2.0.54。
